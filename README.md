## Description

[admin-tempate](https://github.com/clianor/admin-template) 은 nestjs와 nextjs로 구성된 Gql 베이스 관리자 사이트 템플릿입니다.


## Installation

```bash
$ yarn ci
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
